# -Practo-Doctor-Data-Web-Scraping-EDA
Scraped 1,002 doctor profiles from Practo across 17 Indian cities using Selenium &amp; Python. Dataset covers specializations, fees, ratings, experience &amp; working hours. EDA reveals Bangalore leads in doctors, Delhi has highest fees (₹1,387), and 38% hold perfect ratings — suggesting platform bias. Python · Pandas · Matplotlib · Jupyter Notebook
